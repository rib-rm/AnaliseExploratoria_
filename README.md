## Atividade Parte 2

Definições:

Média: Medida que demonstra a concentração dos dados em torno de um valor.
Corresponde à soma dos valores de uma variável, dividida pela quantidade de observações.

Mediana: A mediana é o valor que divide os dados em duas partes iguais.
Organizamos os dados em ordem crescente e selecionamos o valor central.
Se houver um número par de dados, a mediana será a média dos dois valores centrais.
Ao contrário da média, a mediana não é afetada por valores muito altos ou baixos.

Variância: A variância é uma medida que indica o quanto os dados estão dispersos em relação à média.
Calculamos da seguinte forma, primeiro subtraímos a média de cada valor, elevamos ao quadrado,
e depois fazemos a média desses quadrados. Quanto maior a variância, mais os dados estão espalhados,
quanto menor, mais próximos da média estão.

Desvio-Padrão : O desvio padrão é uma medida que indica o quanto os dados se afastam, em média, da média.
Ele é a raiz quadrada da variância. Se o desvio padrão for baixo, significa que os dados estão
mais próximos da média. Se for alto, significa que os dados estão mais espalhados.

Amplitude: A amplitude é a diferença entre o maior e o menor valor de um conjunto de dados.
Ela mostra o intervalo total dos dados. Quanto maior a amplitude, maior é a variação
entre o valor mais baixo e o mais alto.
